## If you're seeing a black screen:

### 1. Force Refresh the Page
- **Windows/Linux**: Ctrl + F5 or Ctrl + Shift + R
- **Mac**: Cmd + Shift + R
- **Mobile**: Pull down to refresh or clear browser cache

### 2. Clear Browser Cache
1. Open browser settings
2. Clear browsing data
3. Select "Cached images and files"
4. Clear data and reload the page

### 3. Try Incognito/Private Mode
Open the link in an incognito/private browser window to bypass cache

### 4. Check Console for Errors
1. Right-click → Inspect
2. Go to Console tab
3. Look for any red error messages
